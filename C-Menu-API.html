      C-Menu Documentation

      This is the main file for C-Menu, a terminal-based application launcher and menu system.

      Note
      It operates by reading a menu description file and displaying a navigable menu in the terminal. Users can select applications to launch or scripts to execute.
      Several ancillary files provide supporting functionality, including handling terminal initialization IO settings, managing the menu structure, and rendering the interface using the NCurses library.
