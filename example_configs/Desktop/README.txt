The script in this directory, install_desktop.sh, installs

cmenu_ghostty.desktop,
cmenu_kitty.desktop, and/or
cmenu_Alacritty.desktop

in your "$HOME"/Desktop directory and the Decision.svg icon in 
/usr/share/pixmaps. When selected, these desktop files will start the terminal
emulator specified, which will then start C-Menu Menu.

If you prefer your own icon, edit desktop file and replace Decision.svg with 
your icon. Feel free to make it your own.
