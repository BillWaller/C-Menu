# Community Code of Conduct

## Our Pledge
We pledge to foster an open, welcoming, diverse, and harassment-free community.

## Our Standards
Examples of positive behavior include empathy, respect, constructive feedback, and accountability. Unacceptable behaviors include harassment, personal attacks, and inappropriate content.

## Enforcement
Maintainers are responsible for enforcing these standards and may take action against violations.

## Scope
This code applies to all community spaces and official representations of the project.
