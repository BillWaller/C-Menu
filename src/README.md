# CMenu Application

CMenu is a customizable command-line menu application that allows users to create and manage interactive menus for various tasks. It is designed to be flexible and easy to use, making it suitable for both beginners and advanced users.

## Features

- Create and manage multiple menus
- Customize menu options and actions
- Support for sub-menus
- Easy navigation using keyboard inputs
- Configurable appearance and behavior
- Cross-platform compatibility
- Open-source and free to use
